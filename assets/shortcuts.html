Help

MotePad Help⌘?

Open a menu from the keyboard

File⌃⌥F
Edit⌃⌥E
Format⌃⌥O
View⌃⌥V
Help⌃⌥H
In any open menu the shortcut is shown to the right of each command. Format's access key is O (not F) to avoid clashing with File — the classic Notepad mnemonic.
